isTaskTime()

Cron.php : 58

Comprueba que una tarea se puede ejecutar según el cron.

public static isTaskTime( $time : int , $cron : array ) : bool

Para cada tarea guardo la programación en variables separadas, representando cada una, una parte del cron, con el siguiente formato:.

    .--------------- minuto (0-59)
    |  .------------ hora (0-23)
    |  |  .--------- día del mes (1-31)
    |  |  |  .------ mes (1-12) o jan,feb,mar,apr,may,jun,jul... (meses en inglés)
    |  |  |  |  .--- día de la semana (0-6) (domingo=0 ó 7) o sun,mon,tue,wed,thu,fri,sat (días en inglés)
    |  |  |  |  |
    *  *  *  *  *

A continuación se define un array con el formato de cada variable, para extraerlo de la hora actual y poder hacer la comparación.
